    "PLAYIT NOW" Makes it easier to accept www.playit.pk as an alternative to www.youtube.com. Now access youtube videos with only 1 extra click than it would take if www.youtube.com were accessible.


    Salient Features
    •     Captures Youtube URLs from Google search results: Now you won't have to go through the trouble of all those extra clicks and copying/pasting of links from here to there
    •     Captures Embedded Youtube videos in webpages: Achieve riddance from situations where your web-surfing is barred by that greyed-out embedded youtube video
    •     Manual video ID entry option: Came across a rare kind of youtube URL (e.g; http://youtu.be/Zs5pys2uwbQ) that was not captured? No worries. Just copy that 11 characters long youtube video ID (found in every youtube URL) and use the manual entry feature
    •     Directly search on www.playit.pk: If nothing else, you can search youtube videos directly, using the search option

    Additional Tips
    •     Use keyboard shortcut to trigger this extension: You can use the Keyboard shortcut, 'Ctrl+Y' ('Command+Y' for MAC), to activate this extension. You can also change the key combination using the UI in chrome://extensions -> Keyboard Shortcuts (scroll to the end)
    •     Press Enter to press buttons: You can use the 'Enter' Key to trigger the respective button in the currently focused section inside the extension
    •     Automatic clipboard checking: This extension will automatically look-up for valid youtube video IDs (e.g; Zs5pys2uwbQ) if copied onto your clipboard and paste it for you in the 'Video ID entry' section
